Input parameters
Form Description
Magnetic Induction In tesla. Positive float number.
Rotation frequency In hertz. Positive float number.
Resistance In ohm. Positive float number.
Contour area In square meters. Positive float number.
End time In seconds. Affects the graph scale. Positive float number.
Steps count Number of points to count function values in them. The more steps, the better the accuracy, but the lower the performance. Positive integer number.

Main forms are designed for the technical task of modeling.

Optional settings are available in the form of a drop-down menu. They contain parameters that were not in the technical task, but that are involved in the construction of the graphics (more fine-tuning)


Use checkbox to choose to draw graphs on one canvas or separately

Explore the graph forms conveniently and efficiently using the menu in the top right corner in graph form.

Legend allows to "disable" one of the plots (EMF or induced current) by clicking.


If you try to enter incorrect data, this field will return to the default value
