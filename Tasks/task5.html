Input parameters
Form Description
R0 Plate radius. In meters. Positive float number.
d Distance between the plates. In meters. Positive float number.
U0 Voltage amplitude. In volts. Positive float number.
ω Angular velocity. In 1/radian. Positive float number.
r Radial coordinate r for second graph. In meters. Positive float number.
Steps count Number of points to count functions values in them. The more steps, the better the accuracy, but the lower the performance. Positive integer number.
End r In meters. Affects the first graph scale. Positive float number.
End time In seconds. Affects the second graph scale. Positive float number.

Main forms are designed for the technical task of modeling.

Optional settings are available in the form of a drop-down menu. They contain parameters that were not in the technical task, but that are involved in the construction of the graphics (more fine-tuning)


Explore the graph forms conveniently and efficiently using the menu in the top right corner in graph form.

If you try to enter incorrect data, this field will return to the default value. Integer types are given with integer type even if a real number is entered
