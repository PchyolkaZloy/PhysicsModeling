Input parameters
Form Description
Charge In coulombs. Can be negative. Float number.
Magnetic induction Defined by a radius vector with coordinates (x, y, z) separated by spaces. Coordinates are float numbers.
Velocity of charge Defined by a radius vector with coordinates (x, y, z) separated by spaces. Coordinates are float numbers.
Flight time In seconds. Affects the chart scale. More seconds - more charge "flies" in the space on the graph. Float number.
Steps count Number of steps for solving differential equations. The more steps, the better the accuracy, but the lower the performance. Integer number.
Mass of charge In kilograms. Real number.

Main forms are designed for the technical task of modeling.

Optional settings are available in the form of a drop-down menu. They contain parameters that were not in the technical task, but that are involved in the construction of the graphics (more fine-tuning)


Explore the graph forms conveniently and efficiently using the menu in the top right corner in graph form.

Legend allows to "disable" one of the plots (charge motion or magnetic induction) by clicking.


If you try to enter incorrect data, this field will return to the default value
