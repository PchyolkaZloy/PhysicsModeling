Input parameters
Form Description
Radius In meters. Positive float number.
Turns amount Positive integer number.
Amperage In ampere. Float number.
Points count In seconds. Affects the graphic image accuracy. The more points, the better the accuracy, but the lower the performance. Float number.
X dist In meters. Affects the scale of the graph. Displays which area from X=0 is displayed on the graph. The default values are selected by radius. The coils are arranged at +- (X dist / 2). Positive float number.

Main forms are designed for the technical task of modeling.

Optional settings are available in the form of a drop-down menu. They contain parameters that were not in the technical task, but that are involved in the construction of the graphics (more fine-tuning)


Explore the graph forms conveniently and efficiently using the menu in the top right corner in graph form.

Legend allows to "disable" one of the plots by clicking.


If you try to enter incorrect data, this field will return to the default value
